std::string CTypeElement::getFormattedMapping ( const CTypeElement *  predecessor) const
protected

the mapping associated to the current type and its predecessor

The meaning of predecessor depends on the TypeElement type: e.g. for a component instance, the predecessor is the ComponentType; for a ComponentType, the predecessor is its base type.

The predecessor's mapping comes first, then the current type's mapping.

Parameters
[in]predecessorA pointer to the predecessor or NULL.
