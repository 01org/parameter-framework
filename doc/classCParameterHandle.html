Detailed Description

Pre v3.1 Handle to a parameter.

Kept for retro-compatibility.

Deprecated:
This class has been deprecated in favor of ElementHandle, Use ElementHandle instead of CParameterHandle. It is a api retro-compatible superset of the pre v3.1 CParameterHandle class.
Note
Achieve retro-compatibility with a derived class instead of an alias (typedef ElementHandle CParameterHandle) as the later would break client compilation in case of client forward declaration.
