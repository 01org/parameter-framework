const char** PfwCriterion::values

Null terminated list of criterion value names.

Example: 

* { "Red", "Green", "Blue", NULL }
*

For an exclusive criterion, the list must not contain more elements then INT_MAX. For an inclusive criterion, the list must not contain more elements then sizeof(int) * BIT_CHAR - 1. Ie: (int)1 << n must not overflow (UB), were n is the number of element in the list.
