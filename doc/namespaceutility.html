Fill a buffer from a file descriptor.

A wrapper around unistd::read that resumes read on incomplete access and EAGAIN/EINTR error.

See also
man 2 read for the parameters.
Returns
true if the buffer could be completely fill, false on failure (see read's man errno section).

If the buffer could not be filled due to an EOF, return false but set errno to 0. Add a custom strerror to prevent logging "success" (sterror(0)) on EOF errors ?
