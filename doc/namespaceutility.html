std::remove_reference<Destination>::type utility::binaryCopy ( const Source  source)

Raw copy of one variable to another of the same size.

This can be regarder as a reinterpret_cast but does a copy and does not break strict-aliasing rules.

The source and the destination must have the same storage size (e.g. copying a uint8_t into a uint32_t won't compile)
