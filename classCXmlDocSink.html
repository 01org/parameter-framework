Detailed Description

The CXmlDocSink class defines how to use a CXmlDocSource.

The interaction between the xml source and xml sink is defined in the process method of CXmlDocSink. One can subclass CXmlDocSink for different purpose by implementing the doProcess method and then use it with any existing implementation of CXmlDocSource.
