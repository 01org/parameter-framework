Detailed Description

template<class CDefaultElementBuilder>
class CDefaultElementLibrary< CDefaultElementBuilder >

Factory that creates an element given an xml element.

If no matching builder is found, it uses the default builder.

Template Parameters:
CDefaultElementBuilderis the class of the element builder to use if no corresponding builder is found for a given xml element.
