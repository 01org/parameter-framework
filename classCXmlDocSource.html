Detailed Description

The CXmlDocSource is used by CXmlDocSink.

The interaction between the xml source and xml sink is defined in the process method of CXmlDocSink. One can subclass CXmlDocSource for different purposes by implementing the populate method and then use it with any existing implementation of CXmlDocSink.
